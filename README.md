# Chat App

## 📌 Description
This project is a **chat application** that allows real-time messaging between users. It includes features like:
- **User Authentication**
- **Real-time Messaging**
- **Read/Unread Message Status**
- **Chatbot Integration**
- **Message Delivery Indicators**
